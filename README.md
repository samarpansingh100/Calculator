# Scientific Calculator in Python

A modern Scientific Calculator built using **Python** and **Tkinter**, featuring a clean and responsive graphical user interface inspired by modern calculator applications.

## Features

### Basic Calculator Functions

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Decimal number support
* Parentheses support
* Clear (C) button
* Backspace (←) button

### Scientific Functions

* Sine (`sin`)
* Cosine (`cos`)
* Tangent (`tan`)
* Square Root (`√`)
* Logarithm Base 10 (`log`)
* Natural Logarithm (`ln`)
* Pi Constant (`π`)
* Square (`x²`)
* Power (`xʸ`)

### User Interface

* Clean modern design
* Responsive layout
* Resizable window
* Dedicated scientific function panel
* Color-coded buttons:

  * Number buttons
  * Operation buttons
  * Scientific function buttons
  * Large equals button

## Technologies Used

* Python 3.13
* Tkinter (GUI Framework)
* Math Module

### Run the Application

```bash
python scientific_calculator.py
```

No additional packages are required since Tkinter and Math are included with Python.
